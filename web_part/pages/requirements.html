Введение в HTML и CSS

HTML (HyperText Markup Language) — это язык разметки, который используется для создания структуры веб‑страниц. Он состоит из элементов (тегов), которые определяют, как будет отображаться контент в браузере.

Основные теги HTML включают:

  • <h1>–<h6> — заголовки разных уровней;
  • <p> — абзацы текста;
  • <ul> и <ol> — списки (неупорядоченные и упорядоченные);
  • <a> — гиперссылки;
  • <img> — вставка изображений.

CSS (Cascading Style Sheets) отвечает за визуальное оформление HTML‑элементов. С его помощью можно задавать:

  • цвета и шрифты;
  • отступы и поля;
  • расположение элементов на странице;
  • эффекты (тени, скругления и т. д.).

Совместное использование HTML и CSS позволяет создавать красивые и функциональные веб‑страницы, которые корректно отображаются на разных устройствах.
